# Web3 dApp Framework & Smart Contract Templates

A production-ready, scalable framework for building decentralized applications on Ethereum. This repository showcases my expertise in full-stack Web3 development, focusing on developer experience (DX), security, and clean architecture.

## 🛠 Tech Stack
- **Smart Contracts:** Solidity, Hardhat, OpenZeppelin
- **Frontend:** React.js, Next.js, Web3.js, Ethers.js
- **Wallet Integration:** MetaMask, WalletConnect
- **Testing:** Chai, Waffle, Hardhat Network

## 📂 Project Highlights
- **Gas-Optimized Contracts:** Implemented efficient storage patterns and access control (Ownable, Roles).
- **Reusable Frontend Components:** Modular React hooks (`useContract`, `useWallet`) for seamless dApp integration.
- **CI/CD & Testing:** 95%+ test coverage with automated deployment pipelines.

##  Getting Started
1. Clone the repository: `git clone https://github.com/Leo-sn1p3rs/web3-dapp-framework.git`
2. Install dependencies: `npm install`
3. Compile contracts: `npx hardhat compile`
4. Run tests: `npx hardhat test`
5. Start frontend: `npm run dev`
